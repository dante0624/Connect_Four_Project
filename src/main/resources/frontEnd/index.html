How to Use Solver

In this game, yellow always plays first and red plays second. The numbers above each column indicates who will win if the current player places their chip in that slot.

A positive number indicates that yellow will win, a negative number indicates that red will win, and 0 indicates a draw. So +5 means that yellow will win in five turns, -4 means that red will win in four turns, and 0 is a draw (which always happens when the board is full).

It also states which color can win and in how many moves, assuming perfect play from both players.
